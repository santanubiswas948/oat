What is LongTerm UI?

LongTerm UI is a minimal, semantic-first UI component library with zero dependencies. It uses modern CSS features like cascade layers, container queries, and CSS custom properties.

How do I use it?

Just include the CSS and JS files, then write semantic HTML. Most elements are styled by default without needing classes.

Is it accessible?

Yes! LongTerm UI prioritizes semantic HTML and ARIA attributes for accessibility. Keyboard navigation is fully supported.
